About My[System]

As I said above, I have a few friends with systems, and one of them showed me the PluralKit Discord bot, and after looking into the PluralKit API, I wanted to create a website for viewing information on your system in an easy-on-the-eyes format, without the need to run through a bunch of the PluralKit commands. In the future, I hope to expand this to include a way to add, edit, and delete members of your system. As well as this, I want to create a mobile app that allows you to add switches on the go, even when not using Discord to help keep track of who is fronting when and how often.

Of course, for roleplayers that use PluralKit, the use cases for My[System] are similar, allowing for a way to view characters of yours and others as well as their descriptions, pronouns, etc.

All that is needed to view a system is the Discord ID of one of the Discord Users linked to the system, which can be found by enabling Developer Mode and right clicking on a user. Once I get a better grasp on the Discord API, it may be possible to search for systems by Discord name and discriminator (#????) alone.

This website is open source and available on GitHub! If you have any suggestions or would like to fix my horrible UI please go ahead and do that on GitHub.
